# Whisper Transcriber (Whisper-Only)

This module is a standalone transcription pipeline under `01-collection/whisper_transcriber`.

It transcribes by downloading temporary audio with `yt-dlp` and running **local Whisper** via `faster-whisper`.

Only lightweight transcript outputs (e.g., `.vtt`, `.txt`) are intended to be stored in-repo.
Raw audio/video files are temporary and should remain out of git.

## What it produces

For each input CSV (e.g., `data/input/mimiminu.csv`), outputs are written to:

`data/<person>/transcripts/`

Files per video:

- `<video_id>.<lang>.source-whisper.vtt` (generated by Whisper)
- `<video_id>.<lang>.txt` (plain text transcript)

## Install (module-local uv)

From repository root:

### 1) Install uv

macOS:

```bash
brew install uv
```

Linux:

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Windows (PowerShell):

```powershell
irm https://astral.sh/uv/install.ps1 | iex
```

Verify:

```bash
uv --version
```

### 2) Install yt-dlp (required)

macOS:

```bash
brew install yt-dlp ffmpeg
```

Fedora:

```bash
sudo dnf install -y yt-dlp ffmpeg
```

Ubuntu/Debian:

```bash
sudo apt-get update
sudo apt-get install -y yt-dlp ffmpeg
```

### 3) Create venv + sync deps

```bash
uv venv 01-collection/whisper_transcriber/.venv
uv sync --directory 01-collection/whisper_transcriber
uv run --directory 01-collection/whisper_transcriber python -m whisper_transcriber.cli --help
```

## Quick start

```bash
uv run --directory 01-collection/whisper_transcriber python -m whisper_transcriber.cli \
  --manual-csv data/input/mimiminu.csv \
  --lang ko \
  --model small \
  --compute-type int8
```

Defaults are chosen to work well on low-VRAM GPUs (e.g., GTX 1050 Ti): `model=small`, `compute-type=int8`.

## Korean accuracy (recommended)

For research-quality Korean transcription, use `large-v3` when possible. On Mac (Apple Silicon), run CPU mode:

```bash
uv run --directory 01-collection/whisper_transcriber python -m whisper_transcriber.cli \
  --manual-csv data/input/mimiminu.csv \
  --lang ko \
  --model large-v3 \
  --device cpu \
  --compute-type int8
```

Note: `large-v3` is substantially slower and uses more RAM than `small`.

## Window progress bar (recommended for long runs)

Enable a single tqdm bar for each window (pre/cancel/post). This tracks transcription time only.

```bash
uv run --directory 01-collection/whisper_transcriber python -m whisper_transcriber.cli \
  --manual-csv data/input/mimiminu.csv \
  --lang ko \
  --model large-v3 \
  --progress-window
```

```bash
  --lang ko --model large-v3 --device cuda --compute-type float16
```

## CLI options

- `--manual-csv`: CSV with `window,video` (same format as `youtube_scraper`)
- `--out-dir`: transcript output directory (default `data/<person>/transcripts`)
- `--tmp-dir`: temp directory for audio downloads (default `data/<person>/.tmp_audio`)
- `--lang`: preferred subtitle language (default `ko`)
- `--model`: Whisper model size (default `small`)
- `--device`: `auto`, `cuda`, or `cpu` (default `auto`)
- `--compute-type`: e.g., `int8`, `int8_float16`, `float16` (default `int8`)
- `--keep-audio`: do not delete temp audio (debugging)
- `--progress-window`: show per-window tqdm progress for transcription

## Notes

- The module calls `yt-dlp` as a subprocess for subtitle/audio download. Install `yt-dlp` system-wide or reuse the one from `01-collection/youtube_scraper/.venv/bin/yt-dlp`.
- Whisper transcription is fully local and free (no API keys).
